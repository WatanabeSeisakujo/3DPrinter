3DPrinter
=========

これは渡辺製作所で3Dプリンターを作ろうというプロジェクです。

## 3Dプリンタの種類

* 光造形
* インクジェット方式
* 粉末石膏造形
* 粉末焼結造形
* 熱溶融積層造形

### 熱溶融積層造形
[RepRap](http://reprap.org/wiki/RepRap/ja) などの台頭により人気を集めている方式。

### 光造形
従来は紫外線レーザーを使うものであり、高価だったが DLP を使ったものが現れた。この方式なら自作も可能である。

[シーフォース株式会社](http://www.digitalwax.asia)が気になる
